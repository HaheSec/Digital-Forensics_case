# PenTest Project

Identified system vulnerabilities in a virtual lab environment and documented findings.
Vulnerability Scan Lab

# Tools Used
- Kali Linux
- Nmap
- Nessus
- VMware

# Objectives
- Perform network scanning and enumeration
- Identify vulnerabilities
- Document and report findings

# Steps Performed
1. Network discovery using Nmap
2. Vulnerability scan with Nessus
3. Analyze results and prioritize risks
4. Write remediation recommendations
